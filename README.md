# Heroes-of-Hammerwatch-II-TChinese
Heroes of Hammerwatch II 繁體中文翻譯

該文本基於遊戲發售日(2025/01/15)所發布的版本製作

目前道具及裝備名稱不會特別依遊戲內容作潤色，只有道具的效果描述會特別潤色。
